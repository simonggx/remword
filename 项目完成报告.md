## 🏗️ 技术架构

### 文件结构
```
remword/
├── manifest.json              # Manifest V3配置
├── scripts/                   # 核心脚本层
│   ├── background.js         # 服务工作者
│   ├── content.js           # 内容脚本
│   ├── database.js          # 数据库管理
│   └── translation.js       # 翻译服务
├── popup/                     # 弹窗界面层
│   ├── popup.html           # 弹窗页面
│   ├── popup.css            # 弹窗样式
│   └── popup.js             # 弹窗逻辑
├── options/                   # 选项页面层
│   ├── options.html         # 选项页面
│   ├── options.css          # 选项样式
│   ├── options.js           # 选项逻辑
│   └── practice.js          # 练习系统
├── styles/                    # 样式层
│   └── content.css          # 内容脚本样式
└── icons/                     # 资源层
    └── [图标文件]
```

### 核心技术栈
- **前端**: HTML5, CSS3, JavaScript ES6+
- **存储**: IndexedDB (Web SQL替代方案)
- **API**: 多翻译服务集成 (MyMemory, LibreTranslate)
- **架构**: Chrome Extension Manifest V3
- **样式**: 响应式设计 + 深色模式支持
